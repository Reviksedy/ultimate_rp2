Rev's Ultimate RP 2 - ресурс-пак, предназначенный для расширения выбора в декоре и косметических предметах. Среди новых предметов есть пропы, шапки, оружие, еда и т.д. Rev's Ultimate RP 2 is a resourcepack that is made to expand the choice in deco and cosmetic items. New items include props, hats, weapons, food, etc.

Установка и использование Installation and usage

Ресурс-пак работает без модов на новейших версиях. Скачав ресурс-пак, поместите его в папку resourcepacks в корневой папке Minecraft, а затем включите его в настройках игры.

Переименуйте нужный предмет в наковальне для получения новой текстуры/модели (букву "ё" можно заменять на "е"). Символ * после или до названия предмета говорит о возможности добавить туда дополнительный текст при желании. (такого функционала ещё нет) The resourcepack works without mods on latest versions. After downloading it, put it into the resourcepacks folder in your game root directory and enable it in the game settings.

Rename an item on an anvil to get a new texture/model. The * symbol after or before the item name suggests that you can add text there if you wish. (can't do that right now)
Предметы, предназначенные для размещения в мире, помещаются в рамку. Props that are meant to be placed in the world should be put in Item Frames.

Разрешается: You are allowed to:

    • Использовать ресурс-пак в видео
    • Использовать ресурс-пак на своём сервере/для строительных проектов
    • Изменять ресурс-пак для личного использования     • Use the resourcepack in your videos
    • Use the resourcepack on your server/building project
    • Edit the resourcepack for personal use

Запрещается: You are NOT allowed to:

    • Выдавать себя за автора
    • Красть текстуры/модели с отсутствием изменений/незначительными изменениями     • Take the authorship
    • Steal textures/models without any changes/with overly minor changes
