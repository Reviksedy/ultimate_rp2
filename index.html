Rev's Ultimate RP 2 - ресурс-пак, изначально сделанный для сервера Realm+ и предназначенный для расширения выбора в декоре и косметических предметах. Среди новых предметов есть пропы, шапки, оружие, еда и т.д.Rev's Ultimate RP 2 is a resourcepack originally made for Realm+ (an SMP) and designed to expand the choice in deco and cosmetic items. New items include props, hats, weapons, food, etc.

Установка и использованиеInstallation and usage

Для использования ресурс-пака нужен OptiFine или CIT Resewn (рекомендуется для игнорирования регистра символов у русских названий), иначе он работать не будет. Скачав ресурс-пак, поместите его в папку resourcepacks в корневой папке Minecraft, а затем включите его в настройках игры.

Переименуйте нужный предмет в наковальне для получения новой текстуры/модели. Символ * после или до названия предмета говорит о возможности добавить туда дополнительный текст при желании. You need to install OptiFine or CIT Resewn for the resourcepack to work. After downloading it, put it into the resourcepacks folder in your game root directory and enable it in the game settings.

Rename an item on an anvil to get a new texture/model. The * symbol after or before the item name suggests that you can add text there if you wish.
